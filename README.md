# Text Based Battleships
A text based version of Battleships I made for a university project. This is written in C++ and my first finished coding project.

---

## Features
- 2 player battleships game (or 1 player by themself).
- Handling incorrect inputs from the user, reprompting them to enter a different answer.
- Detecting if inputted start point and end point of each ship is in horizontal or vertical line.
- Keeps track of how many ships from each player has been destroyed.
- Some sound effects.

---

## Improvements
- A web version that is playable in browser.
- The game opens in separate windows for each player.

---

## Running The Project
Open the solution (.sln) file in any IDE you prefer and press start to run the project in the console.
